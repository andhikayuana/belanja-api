## Simple Product API for Belanja

### Requirements

* Git
* Composer
* PHP 7 or latest with SQLite PDO

### Installation

* Clone this repository using `git`

```bash
git clone git@github.com:andhikayuana/belanja-api.git
```

* Install dependencies using `composer`

```bash
cd belanja-api
composer install
```
